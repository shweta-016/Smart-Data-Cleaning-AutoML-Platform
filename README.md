# Smart Data Cleaning and AutoML Web Platform

Smart Data Cleaning and AutoML Web Platform is a machine learning project developed using Flask, Python, and Scikit-learn. The platform allows users to upload CSV datasets, automatically clean and preprocess the data, apply machine learning models with hyperparameter tuning, compare model performance, and download the cleaned dataset through a professional web interface.

## Features
- Upload CSV datasets
- Automatic data cleaning
- Missing value handling
- Duplicate removal
- Outlier treatment
- Data preprocessing and encoding
- Automatic classification/regression detection
- AutoML model selection
- Hyperparameter tuning using GridSearchCV
- Model performance comparison with charts
- Download cleaned dataset
- Professional web dashboard

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- HTML
- CSS
- Bootstrap
